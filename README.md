# 🛡 AI SOC Dashboard

An AI-powered Security Operations Center (SOC) Dashboard built using **React**, **Vite**, **Express**, and **Google Gemini AI**. The dashboard enables SOC analysts to monitor alerts, analyze logs, visualize security analytics, and interact with an AI assistant for cybersecurity-related queries.

---

## 🚀 Features

- 📊 Interactive Dashboard
- 🚨 Security Alerts Management
- 📜 System Logs Viewer
- 📈 Security Analytics
- 🌍 Threat Intelligence
- 🤖 Gemini AI Security Assistant
- 📥 Export Logs to CSV
- 🔔 Notifications Panel
- ✉️ Messages Panel
- 👤 User Profile Menu
- ⚙️ Settings Page

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/skoduri5-ux/AI-SOC-Dashboard.git

cd AI-SOC-Dashboard
```

---

### Install Frontend

```bash
npm install
```

---

### Install Backend

```bash
cd backend

npm install
```

---

## 🔑 Environment Variables

Create

```
backend/.env
```

Example

```env
PORT=5000

GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

---

## ▶️ Run Backend

```bash
cd backend

npm run dev
```

Backend runs on

```
http://localhost:5000
```

---

## ▶️ Run Frontend

```bash
npm run dev
```

Frontend runs on

```
http://localhost:5173
```
